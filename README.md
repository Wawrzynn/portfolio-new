<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repo</p>

```
git clone [template_directory]
```

<p>2. Go to portfolio</p>

```
cd portfolio-new
```

<p>3. Install dependencies</p>

```
npm install
```

<p>4. Run the project</p>

```
npm run dev
```

<p>5. Go to localhost:3000 in web browser</p>

```
http://localhost:3000/ 
```
